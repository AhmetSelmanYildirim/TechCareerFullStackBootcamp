### Docker Commands

```bash
docker-compose up
```

---

### Backend Commands

**backend project path**

```bash
cd ./ApiTodo
```

**install dependencies**

```bash
mvn clean install
```

**run project**

```bash
mvn spring-boot:run
```

---

### Frontend Commands

**frontend project path**

```bash
cd ./web-todo
```

**install dependencies**

```bash
yarn install
```

**run project**

```bash
yarn dev
```
